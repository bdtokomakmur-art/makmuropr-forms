⚡ Setup Real-time Dashboard

Dashboard ini terhubung ke Google Sheets via Google Apps Script.
Ikuti 3 langkah berikut — sekali setup, dashboard auto-refresh selamanya.

1
Buka Google Apps Script
Buka link berikut di tab baru (sudah terhubung ke Spreadsheet Toko Makmur):
script.google.com → New Project (atau Extensions → Apps Script dari Google Sheets)
2
Paste kode & Deploy sebagai Web App
• Hapus kode default → paste isi file arco-gas-api.js yang sudah didownload
• Klik Deploy → New Deployment → Web App
• Execute as: Me | Who has access: Anyone
• Klik Deploy → Authorize → Copy URL yang muncul
3
Paste URL Web App di sini
💾 URL disimpan di browser (localStorage) — tidak perlu input ulang setelah reload.
